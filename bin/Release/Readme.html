Tumblr Backup (Windows only)

Thank you for using the Tumblr Backup GUI!

All the credit for this utility goes to bbolli for creating the tumblr_backup Python script which does the heavy lifting. All I did was package it up for ease of use.


This script downloads and archives multiple blogs and outputs the results in the desired folder. Each blog gets it's own folder inside the destination folder.

I put this together over a couple of days, and I can't guarantee it's bug free. If the GUI is not working for you, you can still use bbolli's original script in the tumblr_backup directory via the windows command line.

I've included a copy of Python in the directory, so if you don't have it or don't know how to install it, don't worry, it's taken care of. You can open the command prompt by holding shift and right clicking the directory, then selecting 'open command window here'.

Enter the following command to run the script:

tumblr_backup.exe blog-name blog-name blog-name ... etc.

There are actually many things that the original Python script can do, which are not available in the simplified GUI interface.

If you're interested I recommend checking out the readme for that script on GitHub to find out all the different options which are available.
